## 1) 프로젝트 설명

| **게임 이름** | Pong Simulator |
| :--- | :--- |
| **플랫폼** | Windows |
| **장르** | 2D 물리 시뮬레이션 |
| **소개** | "DirectX11과 WinAPI를 사용하여 만든 간단한 Pong 스타일의 물리 시뮬레이터. |

<br>

## 2) 프로젝트 목표
- DirectX11을 활용한 그래픽 렌더링 학습
- 게임 루프 설계 및 프레임 관리 학습

<br>

# 2. 기술스택

- **프로그래밍 언어**: C++14
- **개발 도구**: Visual Studio 2022
- **API**: WinAPI, DirectX11
